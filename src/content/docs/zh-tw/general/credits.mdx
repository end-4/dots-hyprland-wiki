## 鳴謝

:::tip[感謝]{icon="heart"}
若沒有以下貢獻者，此專案無法走到今天這一步。
:::

**辛勤的開發者們：** _請參閱上一節。_

**熱情的貢獻者們：**
- [@midn8hustlr](https://github.com/midn8hustlr) 大幅改善了色彩生成系統，並實作了工作區群組功能。
- [@ZeyadMoustafaKamal](https://github.com/ZeyadMoustafaKamal) 引入 `uv` 為 Python 相關功能建立虛擬環境。[^1]
- [@xiaomeng2004](https://github.com/xiaomeng2004) 在 AGS 和 Quickshell 版本的 illogical-impulse 中實作多語言支援。
- [@Makrennel](https://github.com/Makrennel) 為 Arch Linux 引入帶有本地 PKGBUILD 的 meta 套件。
- [@moetayuko](https://github.com/moetayuko) 提交了許多修正 AGS / Hyprland / Quickshell 設定的 PR。
- 跨發行版支援：
  - Gentoo：由 [@jwihardi](https://github.com/jwihardi) 引入。
  - Fedora：由 [@ririko6z](https://github.com/ririko6z) 引入。
    - 另外 [@EisregenHaha](https://github.com/EisregenHaha) 曾維護 Fedora 的 fork 長達約 9 個月。
- 以及[許多其他人](https://github.com/end-4/dots-hyprland/graphs/contributors)（還有[文件網站的貢獻者](https://github.com/end-4/dots-hyprland-wiki/graphs/contributors)），雖然上面沒有列出他們的名字，但他們也花時間讓這個專案變得更好。

- AGS：[Aylur](https://github.com/Aylur/dotfiles/tree/ags-pre-ts)、[kotontrion](https://github.com/kotontrion/dotfiles)。

**最後但同樣重要的是：**

- 靈感來源：osu!lazer、Windows 11、Material Design 3、AvdanOS（概念）
- 複製：只要遵循授權條款（GNU GPL v3.0），重新發布和重新構建當然是可以的。
