## Чего ожидать?

- **Qt-приложения, а не GTK**. Последние хотя и поддерживают тему, не используются по умолчанию.
- **Material Design 3**.
- **Знакомые бинды** для тех, кто перешел с Windows или GNOME. Нет сочетаний из Wim.
- **Ограничения**: В случае с dotfiles, все и так понятно.
