## Чого очікувати

- **Перевага надається додаткам Qt** над GTK. Останні все ще мають належну тему, але не використовуються за замовчуванням.
- **Зовнішній вигляд у стилі Material Design 3**
- **Звичні комбінації клавіш** для тих, хто переходить з Windows або GNOME. Ніяких дивних комбінацій клавіш з vim!
- **Суб'єктивність**: Це само собою зрозуміло для дотфайлів
